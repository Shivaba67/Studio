# Shivba's Studio – Sidebar Navigation UI

## Overview
This project is a **simple sidebar navigation UI** created using **HTML and CSS**. The sidebar opens and closes using a **checkbox toggle technique**, so no JavaScript is required.

## Features<br>
- Slide-in sidebar navigation<br>
- Hover effects on menu items<br>
- Social media icons<br>
- Background image layout<br>
- Font Awesome icons integration<br>
  
## Technologies Used<br>
- HTML5<br>
- CSS3<br>
- Font Awesome<br>
